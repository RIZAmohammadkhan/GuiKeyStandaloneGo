Your Personal Activity Monitoring Suite Generator!

GenMon's Key Specter-acles:

Craft your own Windows activity monitoring tools with ease! GenMon empowers you to generate a client-server suite designed for effective and secure observation.

  • Effortless Tool Creation: Generate a full monitoring suite (client & server) via a user-friendly GUI â€“ no Go installation required on your system!
  • Stealthy Windows Client: The generated client discreetly tracks application usage and keyboard input, with local encrypted SQLite caching and Windows auto-start capability.
  • Secure P2P Data Sync: Client activity logs are AES-256 encrypted and securely synchronized to your server over a robust P2P (Libp2p) connection.
  • Centralized Log Server: Your dedicated server securely receives, decrypts, and stores all client logs in its own SQLite database, featuring configurable data retention.
  • Simple Web UI for Insights: Easily view and browse collected activity logs from all connected clients through the server's straightforward, built-in web interface.
  • Pure Go & Portable: Both the GenMon generator and the client/server tools it creates are built with pure Go, ensuring clean, portable executables without CGo dependencies.

A Friendly Haunt from the Developers: Please be patient! After the client is first launched, it may take up to 20 minutes for initial data to appear on the server. This is due to the mystical ways of P2P network discovery, routing establishment, and NAT traversal. Once connected, transmissions will be more regular!
